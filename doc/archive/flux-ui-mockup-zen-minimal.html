Plugin UI Influence Map
current surfaces only

Command Palette

All commands are plugin command entries. Some expose pin buttons, parameter steps, input fallback, and recent usage.

Instant Suggestions

Calculator contributes result rows such as 1+2*3 = 7 with copy action.

Workspace Renderer

Text Diff owns the side-by-side renderer and JSON semantic toolbar. Framework hosts the surface.

Panels / Toolbar

Plugins can contribute top-right toolbar buttons and V2 panels. Regex Tester and JS Filter appear as plugin command surfaces, not default chrome.

Pinned Runner

Pinnable live commands become sidebar pinned actions and open the input/output runner view.

Package Manager

Plugin package lifecycle UI shows builtin, installed, dev, read-only source, actions, capabilities, and visible errors.
