策略

迁移重点

优先重构全局 token、sidebar、pane tabs、Command Palette、PluginCard、SourceViewer、PinnedRunner 和 TextDiff toolbar；避免新增框架级 diff API。

验证重点

检查 1280×800 无横向滚动、亮暗主题对比、焦点态、插件来源、错误卡可读性、Monaco diff 仍可编辑。

为什么高级

高级感来自密度控制、边界清晰、状态可见、材质克制和插件信息架构，而不是靠大渐变、巨型 hero 或装饰玻璃卡片。
